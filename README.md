# 📱 센터방문 개통 요청서 웹앱

모바일 최적화된 센터방문 개통 요청서 작성 및 관리 웹 애플리케이션입니다.

## 🌟 주요 기능

- 💼 센터 정보 관리 시스템
- 📝 매출자 정보 템플릿 관리
- 📱 모바일/데스크톱 반응형 디자인
- 💾 자동 저장 기능
- 📄 문서 다운로드
- ⚡ PWA 지원

## 🚀 시작하기

1. 저장소 클론
```bash
git clone [repository-url]
cd center
```

2. 웹 서버 실행
```bash
# Python의 경우
python -m http.server 8000

# Node.js의 경우
npx http-server
```

3. 브라우저에서 열기
```
http://localhost:8000
```

## 💻 사용 방법

1. **센터 정보 설정**
   - 최초 실행시 센터 정보를 설정해주세요
   - 설정된 정보는 자동으로 저장됩니다

2. **매출자 템플릿 관리**
   - 자주 사용하는 매출자 정보를 템플릿으로 저장
   - 드롭다운에서 선택하여 빠르게 입력

3. **문서 작성**
   - 필요한 정보를 순서대로 입력
   - 실시간으로 미리보기 확인
   - 완성된 문서는 텍스트 파일로 다운로드

4. **모바일/데스크톱 모드**
   - 우상단의 모드 전환 버튼으로 레이아웃 변경
   - 모바일에서는 자동으로 최적화된 레이아웃 적용

## 📱 PWA 설치 방법

1. Chrome/Safari에서 웹앱 접속
2. 설치 프롬프트가 표시되면 "설치하기" 클릭
3. 홈 화면에서 앱 아이콘으로 실행 가능

## 🔧 기술 스택

- HTML5
- CSS3 (Grid, Flexbox)
- Vanilla JavaScript
- PWA (Service Worker, Web Manifest)
- LocalStorage
